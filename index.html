JavaScript Loading Strategies
(async and defer)

async allows js files to download as html is parsing and will pause html loading only until excecute js is complete

defer allows js files to download as html is parsing and will execute js after html has finished parsing (loading)

no async and defer: html parsing (loading) is paused until js is downloaded and excecuted (deployed)

JavaScript Outputs

alert("example"); this will pop up an alert in browser which you need to click ok to continue

prompt("example question?", "placeholder answer"); this will pop up an alert in browser which you need to type in an answer and click ok

confirm("are you sure you want to continue?"); this will pop up a message with options ok and cancel
